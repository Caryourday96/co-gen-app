Siemens Co-Gen Power Plant Data Analysis

The Project

The main objective of the project is to evaluate the data from Siemens Co-Gen power plant and visualized the results. In this case the data is generated electricity and carbon dioxide over time. We chose to display the results through this web application because it can be viewed with any device and on any platform.
